#T4G Catch-Up Assignments

This repository contains my Tech4Girls (T4G) catch-up assignments.  
Each folder represents a specific week of learning, practice tasks, and hands-on projects.


 Week 1 – Web Fundamentals

Description 
Introduction to how the web works and core web technologies.

Topics Covered:
- How websites are searched and loaded on the internet

**What I Learned:**
- How browsers communicate with servers
- The role of web servers and application servers


Week 2 – Linux Basics

Description:  
Introduction to Linux commands, terminal navigation, and file system management.

**Topics Covered:**
- pwd, cd, ls, cat
- Absolute vs Relative paths
- Creating and moving files/folders
- Using `echo` with redirection (`>`, `>>`)
- Basic Vim commands

**What I Learned:**
- How to confidently navigate the Linux file system
- The difference between absolute and relative paths
- How to create, edit, and manage files using the terminal
- Improved command-line efficiency and confidence



Week 2 – Git & GitHub

**Description:**  
Understanding version control and collaboration using Git and GitHub.

Topics Covered:
- What is Git
- What is GitHub
- Initializing repositories (`git init`)
- Staging and committing changes
- Branching basics
- Connecting local repo to GitHub

**What I Learned:**
- How version control works
- The difference between local and remote repositories
- How to track changes in a project
- How to push projects to GitHub
- The importance of commits in documenting progress
